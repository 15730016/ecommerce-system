# Hệ Thống E-Commerce

Hệ thống e-commerce gồm 3 module độc lập:

- **laravel-api**: Backend (Laravel + Sanctum)
- **web-app**: Frontend (React + TailwindCSS)
- **mobile-app**: Mobile App (React Native)
